# 导航菜单配置指南

本文档旨在说明如何配置网站前端主导航菜单，包括“简单链接”和“嵌套菜单”（Mega Menu）。

## 菜单结构概览

导航菜单由一系列菜单项组成，每个菜单项可以是以下两种类型之一：

1.  **简单链接**：一个标准的链接，点击后直接跳转到指定页面。
2.  **嵌套菜单**：一个包含复杂布局的下拉菜单，通常用于展示分类、子分类和推广图片。

---

## 1. 配置简单链接

“简单链接”是最基础的菜单类型。

**配置项说明：**

-   **菜单名称**：显示在导航栏上的文本（例如：`Shop`）。
-   **链接地址**：点击后跳转的 URL（例如：`/shop`）。
-   **类型**：选择“简单链接”。

**示例：**

| 菜单名称 | 链接地址 | 类型 |
| :--- | :--- | :--- |
| `Shop` | `/shop` | 简单链接 |

---

## 2. 配置嵌套菜单（Mega Menu）

“嵌套菜单”允许您创建一个包含多列子菜单和图片的丰富下拉菜单，非常适合用于“女士”、“男士”等主分类。

**配置步骤：**

### 第一步：创建主菜单项

首先，创建一个顶级的菜单项。

-   **菜单名称**：显示在导航栏上的文本（例如：`Women`）。
-   **链接地址**：主分类的 URL（例如：`/shop/category/women`）。
-   **类型**：选择“嵌套菜单”。

### 第二步：配置嵌套菜单内容

选择了“嵌套菜单”后，您需要配置下拉菜单的具体内容。

-   **主图（Main Image）**：在下拉菜单中显示的推广图片。这通常是一张吸引人的大图，用于展示该分类的特色。
-   **主标题（Main Title）**：显示在图片旁边的标题（例如：`Women's Collections`）。

### 第三步：添加子菜单项目

在主标题下方，您可以添加具体的子菜单链接。这些链接通常会分列显示。

每个“子菜单项目”包含以下配置：

-   **图片**：显示在子菜单项旁边的小图标或缩略图。
-   **名称**：子菜单项的显示文本（例如：`Crop Tops`, `T-Shirts`）。
-   **链接地址**：点击后跳转的具体分类或产品列表页面的 URL（例如：`/shop/category/women/crop-tops`）。

**示例（以 "Women" 菜单为例）：**

-   **主菜单项:**
    -   **菜单名称**: `Women`
    -   **链接地址**: `/shop/category/women`
    -   **类型**: `嵌套菜单`

-   **嵌套菜单内容:**
    -   **主图**: (上传一张女性模特的图片)
    -   **主标题**: `Women's Collections`

-   **子菜单项目 (部分示例):**

| 图片 | 名称 | 链接地址 |
| :--- | :--- | :--- |
| (Crop Top 图片) | `Crop Tops` | `/shop/category/women/crop-tops` |
| (T-Shirt 图片) | `T-Shirts` | `/shop/category/women/tshirts` |
| (Skirt 图片) | `Skirts` | `/shop/category/women/skirts` |
| (Formal Wear 图片) | `Formal Wear` | `/shop/category/women/formal-wear` |

按照以上步骤，您可以轻松地配置出如效果图所示的导航菜单。
